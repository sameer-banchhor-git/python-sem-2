2. Palindrome Check Easy

Description: A function that takes a string input and checks if it is a palindrome or not.

Rationale: Requires string manipulation (reversing) and comparison. Not too complex.

Python 

def is_palindrome(text):
    processed_text = text.lower().replace(" ", "")
    return processed_text == processed_text[::-1]

# Example usage
word = "Racecar"
if is_palindrome(word):
    print(f"'{word}' is a palindrome.")
else:
    print(f"'{word}' is not a palindrome.")

5. Factorial Calculation Easy

Description: A function that takes an integer input and calculates the factorial of that number.

Rationale: Basic iterative (or recursive) function. Well-defined mathematical concept.

Python 

def factorial(n):
    if n == 0:
        return 1
    else:
        result = 1
        for i in range(1, n + 1):
            result *= i
        return result

# Example usage
number = 5
fact = factorial(number)
print(f"The factorial of {number} is {fact}")

2. Fibonacci Sequence Medium

Description: Write a program to generate the Fibonacci sequence up to a certain number of terms.

Rationale: Requires understanding the Fibonacci sequence and implementing it either iteratively or recursively. Slightly more involved than basic loops.

Python 

def fibonacci_sequence(n):
    fib_list = []
    a, b = 0, 1
    for _ in range(n):
        fib_list.append(a)
        a, b = b, a + b
    return fib_list

# Example usage
num_terms = 10
fib_seq = fibonacci_sequence(num_terms)
print(f"Fibonacci sequence: {fib_seq}")

Description: Write a function that takes an integer n as input and calculates the value of 1 + 1/1! + 1/2! + ... + 1/n!

def factorial(n):
    if n == 0:
        return 1
    else:
        result = 1
        for i in range(1, n + 1):
            result *= i
        return result

    sum = 1.0  # Start with 1 (for 1/0!)

6. Linear and Binary Search Medium

Description: Write a program to implement linear and binary search on lists.

Rationale: Requires understanding the algorithms (linear is easy, binary is a bit more complex) and implementing them correctly. Binary search requires a sorted list.

Python 

def linear_search(list, target):
    for i in range(len(list)):
        if list[i] == target:
            return i  # Return the index if found
    return -1  # Return -1 if not found

def binary_search(list, target):
    left = 0
    right = len(list) - 1
    while left <= right:
        mid = (left + right) // 2
        if list[mid] == target:
            return mid
        elif list[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1  # Return -1 if not found

# Example usage
my_list = [2, 5, 7, 8, 11, 12]  # MUST be sorted for binary search
target_value = 11

linear_result = linear_search(my_list, target_value)
binary_result = binary_search(my_list, target_value)

print(f"Linear search result: Index {linear_result}")
print(f"Binary search result: Index {binary_result}")

1. Stack Class Hard

Description: Write a program to create a Stack class and implement all its methods (using lists).

Rationale: Requires understanding the Stack data structure (LIFO) and implementing it with its core methods (push, pop, peek, isEmpty) using lists. Focuses on object-oriented programming concepts.

Python 

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.isEmpty():
            return self.items.pop()
        else:
            return None  # Or raise an exception

    def peek(self):
        if not self.isEmpty():
            return self.items[-1]
        else:
            return None  # Or raise an exception

    def isEmpty(self):
        return len(self.items) == 0

    def size(self):
        return len(self.items)

# Example usage
my_stack = Stack()
my_stack.push(1)
my_stack.push(2)
my_stack.push(3)

print(f"Stack size: {my_stack.size()}")
print(f"Top element: {my_stack.peek()}")
print(f"Popped element: {my_stack.pop()}")
print(f"Is stack empty? {my_stack.isEmpty()}")

2. Queue Class Hard

Description: Write a program to create a Queue class and implement all its methods (using lists).

Rationale: Similar to Stack, but requires understanding the Queue data structure (FIFO) and its methods (enqueue, dequeue, peek, isEmpty).

Python 

class Queue:
    def __init__(self):
        self.items = []

    def enqueue(self, item):
        self.items.insert(0, item) #insert at beginning of list

    def dequeue(self):
        if not self.isEmpty():
            return self.items.pop() #remove from the end of list
        else:
            return None  # Or raise an exception

    def peek(self):
        if not self.isEmpty():
            return self.items[-1]
        else:
            return None  # Or raise an exception

    def isEmpty(self):
        return len(self.items) == 0

    def size(self):
        return len(self.items)

# Example usage
my_queue = Queue()
my_queue.enqueue(1)
my_queue.enqueue(2)
my_queue.enqueue(3)

print(f"Queue size: {my_queue.size()}")
print(f"Front element: {my_queue.peek()}")
print(f"Dequeued element: {my_queue.dequeue()}")
print(f"Is queue empty? {my_queue.isEmpty()}")

3. Sorting Algorithms Hard

Description: Write a program to sort a list using insertion sort, bubble sort, and selection sort.

Rationale: Requires understanding and implementing three different sorting algorithms. Each has its own logic and requires careful attention to detail to implement correctly. Also a good exercise in understanding algorithm efficiency.

Python 

def insertion_sort(list):
    for i in range(1, len(list)):
        key = list[i]
        j = i - 1
        while j >= 0 and key < list[j]:
            list[j + 1] = list[j]
            j -= 1
        list[j + 1] = key
    return list

def bubble_sort(list):
    n = len(list)
    for i in range(n):
        for j in range(0, n - i - 1):
            if list[j] > list[j + 1]:
                list[j], list[j + 1] = list[j + 1], list[j]
    return list

def selection_sort(list):
    n = len(list)
    for i in range(n):
        min_idx = i
        for j in range(i + 1, n):
            if list[j] < list[min_idx]:
                min_idx = j
        list[i], list[min_idx] = list[min_idx], list[i]
    return list

# Example usage
my_list = [64, 34, 25, 12, 22, 11, 90]

print(f"Original list: {my_list}")

insertion_sorted_list = insertion_sort(my_list.copy()) #using .copy() to not modify original list.
print(f"Insertion sorted list: {insertion_sorted_list}")

bubble_sorted_list = bubble_sort(my_list.copy())
print(f"Bubble sorted list: {bubble_sorted_list}")

selection_sorted_list = selection_sort(my_list.copy())
print(f"Selection sorted list: {selection_sorted_list}")
